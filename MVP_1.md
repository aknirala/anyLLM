# MVP_1: Complete Functionality

This document outlines the requirements for the first complete Minimum Viable Product (MVP).

The landing page is `anyLLM.html`.

## Page Layout

The page will be divided into two main panes:

*   **Left Pane (Side Pane):** This pane will display a list of conversations, similar to the interface of chat applications like ChatGPT or Gemini. See [Side Pane](./MVP_1_doc/3_side_pane.md) for more details.
*   **Right Pane (Main Pane):** This pane will contain the main chat interface and the LLM selection controls. See [Page Skeleton](./MVP_1_doc/1_page_skelton.md) for more details.

## Functionality and Features

The implementation of the features will follow the order of the numbering of the files in the `MVP_1_doc` directory.
